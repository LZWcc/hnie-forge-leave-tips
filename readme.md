## 安装依赖

如需打包， 请在项目根目录下依次执行以下命令，安装项目所需依赖(开发可clone development分支)：

```bash
npm install uview-ui

npm install uview-ui lodash-es dayjs json-stringify-pretty-compact
```

并在 `manifest.json`的 `"appid"`中修改为自己的Appid
